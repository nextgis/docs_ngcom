Демо-проекты
===============

Протестируйте один из наших демо-проектов с готовыми слоями, стилями и картами. Когда вы заберете демо-проект себе, вы сможете его отредактировать, сбросить к начальному состоянию или удалить. Также демо-проект можно открыть в режиме только для чтения. Каждый проект содержит инструкцию по созданию похожего проекта с вашими данными.
